Introduction

This technical documentation represents the organization of the first semester of a First-Year or Introductory Algebra class. It is not really a technical document, but it will help me organize another project while completing this free code camp requirement.

Unit 1

Working with Real Numbers

Review of the arithmetic operations of addition, subtraction, multiplication, and division with integers, decimals, and fractions. This unit also reviews the basic properties of these operations. There is a basic introduction to the set of real numbers and its subsets, numerical and algebraic expressions, powers and exponents, percents, and absolute value. The unit wraps up with a review of order of operations

Unit 2

Algebraic Expressions

This unit introduces the use of letters or symbols to represent numbers, allowing us to create short algebraic expressions to replace longer verbal statements. First, we practice adding and subtracting like terms, also known as combining like terms, to simplify algebraic expressions. Next, we use the distributive property to expand algebraic expressions. This is followed by factoring algebraic expressions, reversing the expansion we did previously. Then we practice evaluating algebraic expressions for given values of the variables. Finally, we practice writing algebraic expressions, going from English to math.

Unit 3

Solving Equations with a Single Variable

In this unit, we will practice solving equations containing a single variable. It will have five sub-units:

  1. Solving equations in one step with addition or subtraction
  2. Solving equations in one step with multiplication or division
  3. Solving equations with two-step
  4. Solving equations with multiple steps
  5. Writing and solving single variable equations
